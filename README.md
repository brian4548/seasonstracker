# Season Tracker

This app uses location services to show certain things pertaining to what season your side of the hemisphere is experiencing(North/South currently). 

Clone the repo and run `npm start`.

Next accept the following pop-up:

![img](https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcSzPCV_9sGxtcgIb5L_CRSYMkeRSa1kERbsSA&usqp=CAU)

If you chose Block then click the `i` in the top right next to your search bar and set it to `Ask by default` and try again.

It the app does not work even when you select `allow` then open your console and find the veritcal `...` at the bottom right. Then select a different city manually.
